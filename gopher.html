A lot to keep track of

We have 1614 Chromebooks to keep track of! We need to keep track of whether it was assinged to a student, who it is assigned to, and how many were issued.

With every device we assign, on Google Admin we input student info, location, and additional notes for the date of assignemnt, as well as any history of previous devieces.

Having received a list of serial numbers of the chromebooks, I generated a data report of all these chromebooks. Here, I am able to see data we inputed on Google Admin.

With the list of serial numbers, I wrote a VLOOKUP function in another sheet to find the user associated with each device serial number.

Any device without the default user, "ptuzzolino@passaic-city.k12.nj.us", is added to a counter that lists the amount of deveices currently assigned and in use, out of the 1614 we have in storage.
