Info Tech - Tools

  • HTML

    • The building block and structure of webpages. Follow along with Professor Spoerri's videos on how to code, as well as reading over his comments. Don't worry about not being able to code from scratch, so long as you grasp the concepts he explains. Particular syntax and coding can be found on W3Schools website for copy and use.

  • CSS

    • A series of statements that identify and effect how content is visually presented. Follow Professor Spoerri's videos as well as read over and copy from W3Schools.

  • PHP

    • A server side language that allows for the output and modification of data. Follow along the Professor's videos and use W3Schools as reference for coding.

  • JavaScript

    • A client-side programming language that allows for the implementation of features within HTML code. Follow along the Professor's videos and use W3Schools as reference for coding.

  • MySQL

    • A web database for websites. Follow along the Professor's videos. Some prerequisites need to be downloaded prior to installation.

  • ScreencastOMatic

    • A free program that allows for recording a user's screen. Follow along the Professor's videos, though the controls are simple enough to play around with. The big red recording/stop button, for example.

  • Flickr

    • Popular photo-sharing platform and social network. Use the search bar to find desired images by keywords.

  • Pixlr

    • Free online photo editor. Follow along with the Professor's videos. Though the controls are also simple. Google helps for additional desired actions.

  • FileZilla

    • Free online file sharing system. Follow along with the Professor.

  • Atom

    • A text editor that includes convenient features to make for a smoother coding experience. Use the same way as something like Notepad. Additional features can be accessed. For example, I wondered if there was an autosave feature for quicker changes of html pages. I googled it and discovered there were numerous other packages and settings one can find if they desired.
